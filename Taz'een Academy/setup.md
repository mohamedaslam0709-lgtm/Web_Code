# 🚀 Getting Started
## Prerequisites
 - A code editor (VS Code recommended).
 - A modern web browser.
 - (Optional) Live Server extension for VS Code.
## Installation
Clone the repository:
```text
git clone https://github.com/mohamedaslam0709-lgtm/Web_Code.git
cd Taz'een-Academy
```
## Open the project:
 - Open the folder in your code editor.
 - Launch the index.html file in your browser, or use a local server (e.g., Live Server) for the best experience.
- **Note:**
**If using VS Code Live Server extension**
- **Right-click index.html -> Open with Live Server**
