# 🌙 Taz'een Qur'an Learning Academy

**Taz'een Academy** is a modern, responsive web platform dedicated to providing authentic and accessible Islamic education. The website serves as a digital gateway for students of all ages to explore Qur'an learning programs, including Tajweed, Hifz (memorization), Islamic Studies, and Spoken Arabic.

Built with a focus on spiritual growth and educational excellence, the academy combines traditional teachings with modern technology to offer flexible one-on-one online sessions.

## ✨ Features

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Modern UI/UX:** Clean interface with a gold and dark theme, smooth scroll animations, and interactive cards.
- **Dynamic Navigation:** Mobile-friendly hamburger menu with smooth transitions.
- **Course Catalog:** Dedicated section showcasing various learning programs (Diploma, Tajweed, Hifz, etc.).
- **Contact & Enrollment:** Integrated contact forms (Netlify-ready) and WhatsApp group integration for community connection.
- **About Section:** Highlights the academy's vision, mission, and commitment to quality education.
- **Privacy Compliance:** Includes a comprehensive Privacy Policy page.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for all pages.
- **CSS3:** Custom styling with CSS Variables (HSL), Flexbox, Grid, and Keyframe Animations.
- **JavaScript (Vanilla):** Intersection Observer API for scroll animations and mobile menu toggling.
- **Font Awesome:** For iconography.
- **Netlify:** Form handling and deployment (forms configured with `netlify` attributes).
